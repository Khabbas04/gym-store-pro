# Gym Store Pro - Laravel Skeleton

A polished Laravel storefront starter for selling athletic shirts.

## Included
- Product model
- Product migration
- Product controller with validation
- Modern Blade + Tailwind CDN design
- Product listing page
- Product details page
- Add product form

## How to use
1. Create a fresh Laravel project:
   composer create-project laravel/laravel gym-store-pro

2. Copy these files into the project.

3. Configure `.env` database values.

4. Run:
   php artisan migrate
   php artisan serve

## Notes
- Images use remote URLs for demo purposes.
- `index()` includes demo fallback cards if the database is empty.
- This is a clean starter, not a complete ecommerce system yet.
